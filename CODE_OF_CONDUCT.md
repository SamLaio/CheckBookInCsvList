# Code of Conduct / 行為準則

> A small project, kept friendly. / 小專案，保持友善。

## English

### Spirit

This is a small userscript project for checking selected book text against a personal library source. We value practical bug reports, readable changes, and good-faith discussion.

### What We Expect

- **Be specific.** "Search does not work" is less useful than "OPDS search against this template returns HTTP 401, and the panel shows an empty result."
- **Be patient.** This is a small project. Response time may vary.
- **Be honest about scope.** This script is not a full library manager, sync service, or browser extension suite.
- **Protect private data.** Do not post real credentials, private OPDS tokens, unpublished CSV URLs, or personal library data in public issues.

### What We Will Not Tolerate

- Personal attacks, harassment, or threats
- Doxxing or sharing private information
- Spam, off-topic promotion, or repeated rule-breaking
- Bad-faith behavior aimed at extracting work from maintainers

### Scope

This applies to GitHub issues, pull requests, discussions, and any communication clearly tied to this project.

## 繁體中文

### 精神

這是一個小型 userscript 專案，用來把網頁上選取的書名或文字拿去比對個人書庫來源。我們重視具體問題、可讀改動與善意討論。

### 我們期待

- **講具體。** 「搜尋壞了」不如「用這個 OPDS template 查詢時回 HTTP 401，面板顯示空結果」有幫助。
- **有耐心。** 這是小專案，回應時間可能不固定。
- **對 scope 誠實。** 這支腳本不是完整書庫管理器、同步服務或瀏覽器擴充套件組。
- **保護私人資料。** 不要在公開 issue 貼真實帳密、私人 OPDS token、未公開 CSV URL 或個人書庫資料。

### 我們不容忍

- 人身攻擊、騷擾、威脅
- 公開他人隱私
- 灌水、不相關宣傳、重複違規
- 惡意榨取維護者勞動的行為

### 範圍

涵蓋 GitHub issues、pull requests、discussions，以及任何明顯跟本專案有關的溝通。
